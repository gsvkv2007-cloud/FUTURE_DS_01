# Business Sales Performance Dashboard

## 📊 Overview
This project presents a **Business Sales Performance Dashboard** built using **Power BI**.  
The dataset used is the **Walmart Sales Dataset** from Kaggle, covering sales data across multiple stores and time periods.  

The dashboard provides insights into sales trends, store performance, and external factors such as fuel prices, CPI, unemployment rates, and holidays.

---

## 📂 Dataset
- **Source:** Walmart Sales Dataset (Kaggle)  
- **Key Features:**
  - Store-level weekly sales  
  - Holiday flag indicator  
  - Economic indicators (CPI, Fuel Price, Unemployment)  
  - Temperature data  

---

## 🛠️ Tools Used
- **Power BI** for data visualization and dashboard creation  
- **Kaggle** dataset for raw data  

---

## 📈 Dashboard Features
- **KPIs:**
  - Total Sales: 6.74 Billion  
  - Total Stores: 45  
  - Average CPI: 171.58  
  - Average Fuel Price: 3.36  
  - Average Unemployment: 8.00%  

- **Visualizations:**
  - Sales by Store (Bar Chart)  
  - Sales by Holiday vs Non-Holiday (Donut Chart)  
  - Sales vs Temperature (Scatter Plot)  
  - Sales vs Fuel Price (Line Chart)  
  - Sales vs CPI (Line Chart)  

---

## 🎯 Insights
- Majority of sales occurred during **non-holiday periods (92.5%)**.  
- **Holiday weeks contributed 7.5%** of total sales, showing seasonal spikes.  
- External factors such as **fuel price and CPI** show correlation with sales trends.  
- Store-level analysis highlights performance differences across 45 locations.  

---

## 🚀 How to Use
1. Clone this repository.  
2. Open the `.pbix` file in **Power BI Desktop**.  
3. Explore the interactive dashboard to analyze Walmart’s sales performance.  

---

## 📌 Conclusion
This dashboard provides a comprehensive view of Walmart’s sales performance, enabling better understanding of:  
- Store-level contributions  
- Impact of holidays  
- Influence of economic indicators  

It serves as a valuable tool for **business analysis and decision-making**.
